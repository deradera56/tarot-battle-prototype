# タロットバトル・デッキ構築ローグライク プロトタイプ

操作感確認用の簡易HTML5プロトタイプ。ブラウザで index.html (または公開URL) を開くだけで遊べます。

## 遊び方
- 下部の手札からカードをタップ → スプレッド(上部5マス)に配置
- 配置済みカードをタップ → 逆位置に反転（HPが30%以下の時だけ解放）
- 配置済みカードを長押し（またはダブルクリック） → 手札に戻す
- 役と予測ダメージがリアルタイムで表示される
- 「攻撃！」で確定 → 敵にダメージ、その後反撃を受ける
- 「テスト:HPを30%にする」ボタンで逆位置モードをすぐ試せる

## 役一覧（企画書と対応）
ワンペア×1.5 / 同スート3枚×2.0 / 階段×3.0 / コートカード役×3.5 / フラッシュ×4.0 / グランドアルカナ×10.0

企画書は `資料/企画書_タロットバトルデッキ構築ローグライク.md` を参照。
